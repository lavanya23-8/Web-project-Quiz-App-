# 🧠 Web Project Quiz App.

A simple and interactive Quiz Application developed using HTML, CSS, and JavaScript. The application allows users to test their knowledge through multiple-choice questions and receive instant feedback and scores..

## 📌 Project Overview.

The Web Project Quiz App is designed to provide an engaging learning experience by presenting users with quiz questions and evaluating their responses. It supports dynamic question loading and score calculation, making it suitable for educational and practice purposes.

## 🚀 Features

- Multiple-choice quiz questions
- Instant answer validation
- Real-time score calculation
- User-friendly interface
- Responsive design
- Supports multiple languages
- Dynamic question loading using JSON files
- Final score display after quiz completion

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- JSON

## 📖 How It Works

1. User opens the quiz application.
2. Questions are loaded from JSON files.
3. User selects an answer for each question.
4. The application checks the answer instantly.
5. Scores are updated automatically.
6. Final score is displayed at the end of the quiz.

## 🌐 Language Support

The application currently supports:

- English
- Kannada

Questions are stored separately in:

- `questions_en.json`
- `questions_kn.json`

## ▶️ How to Run

### Method 1: Run Locally

1. Download or clone the repository.

```bash
git clone https://github.com/lavanya23-8/Web-project-Quiz-App-.git
```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

### Method 2: Using VS Code Live Server

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.
